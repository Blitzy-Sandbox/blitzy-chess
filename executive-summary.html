The Idea

One board. Two ways to play.

A clean, modern chess app in the browser. Take on a smart computer opponent, or play a friend in real time.

  • Play a friend

What We Delivered

Three things, done well.

  • Smart AI Opponent

    A computer rival with three skill levels — from a friendly warm-up to a serious challenge.

  • Real-Time Multiplayer

    Invite a friend with a short code and play live, move for move.

  • Self-Playing Showcase

    The AI plays itself on screen, records the game, and narrates every move.

Pick your challenge.

Deeper look-ahead and more thinking time make each tier noticeably stronger.

Real-Time Multiplayer

Play a friend in seconds.

  • One tap creates a room with a short, shareable 6-character code.
  • Your friend joins with the code — no account, no setup.
  • Moves are relayed live and checked by the server, so the game stays fair.
  • Briefly disconnected? You have 60 seconds to rejoin and pick up right where you left off.

A Built-From-Scratch Brain

We built the chess mind ourselves.

No off-the-shelf engine. Four strengths work together to make strong, human-feeling moves.

  • Opening knowledge

    It starts from proven opening ideas, just like a seasoned player.

  • Position judgment

    It weighs who stands better, and why, on every turn.

  • Deep look-ahead

    It reads many moves ahead to find the strongest plan.

  • Endgame precision

    It plays tight endgames with exact, textbook technique.

The Self-Playing Showcase

The AI puts on a show.

  • The computer plays itself inside the real game screen.
  • Every move holds for at least five seconds, so it is easy to follow.
  • The whole game is captured to a video file.
  • A written, timestamped commentary explains the thinking behind each move.
You walk away with a shareable highlight reel: a recorded game plus a play-by-play transcript.

How It Fits Together

A simple, sturdy shape.

The browser shows the board. The server runs the game and the AI, and has the final say.

Built to Trust

The server is the referee.

  • Every move is checked on the server before it counts.
  • Illegal moves are simply refused.
  • Both players always see the same, correct board.
  • The result is a fair, consistent game every time.

Quality & Visibility

We can see how it is running.

  • Automated tests guard the rules and the gameplay.
  • Structured logs and request tracing show exactly what happened.
  • A live metrics feed and health checks report system status.
  • A ready-made operations dashboard ships with the project.

Easy to Run & Extend

From clone to running, in one command set.

  • A single set of make commands runs everything — setup, play, test, and the showcase.
  • Clear onboarding takes a new developer from a blank machine to a running app.
  • A decision log records why each important choice was made.
  • make init
  • make dev
  • make self-play
  • make test

Technology, Briefly

Modern, proven tools.

  • A Python backend (FastAPI) runs the game and the AI.
  • A React front end draws the board and the controls.
  • Live play travels over a fast, always-on connection (WebSocket).
  • A trusted chess library enforces the rules.

Familiar, well-supported technology — easy to staff and easy to maintain.
